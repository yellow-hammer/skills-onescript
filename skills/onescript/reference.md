# OneScript — справочник (синтаксис и отличия от 1С)

## Отличия от 1С

- Сборка мусора .NET, не подсчёт ссылок. Для явной очистки: `ВыполнитьСборкуМусора()`, `ОсвободитьОбъект()`.

## Формат даты

В функции `Формат` для параметра ДФ добавлен формат `р` (кириллица) или `f` для дробных секунд:

```bsl
Строка = Формат(Дата, "ДФ=ЧЧ:мм:сс.ррр");
// "20:02:53.345"
```

До 6 разрядов дробной части.

## Конструктор в выражении

В OneScript допустимо обращаться к результату конструктора в том же операторе:

```bsl
Если Новый Файл("myfile.txt").Существует() Тогда
    // ...
КонецЕсли;
```

В 1С такое запрещено.

## Параметризованные исключения

Можно передать в исключение объект с дополнительными данными:

```bsl
ВызватьИсключение Новый ИнформацияОбОшибке("Текст ошибки", ДополнительныеДанные);
```

В блоке `Исключение` у `ИнформацияОбОшибке()` будет заполнено свойство `Параметры`.

## Rethrow с ИнформацияОбОшибке

Перевыброс существующего исключения (например, из фонового задания):

```bsl
ОшибкаВЗадании = МоеФоновоеЗадание.ИнформацияОбОшибке;
Если ОшибкаВЗадании <> Неопределено Тогда
    ВызватьИсключение ОшибкаВЗадании;
КонецЕсли;
```

## Загрузчик библиотек

В корне библиотеки можно разместить `package-loader.os` с процедурой `ПриЗагрузкеБиблиотеки(Путь, СтандартнаяОбработка, Отказ)`. Вызовы `ДобавитьКласс(ИмяФайла, Идентификатор)` и `ДобавитьМодуль(ИмяФайла, Идентификатор)` регистрируют типы и модули. Если `СтандартнаяОбработка = Ложь`, встроенный алгоритм загрузки не выполняется.
